# SQL Cockpit v1.0 Released

We're excited to announce the release of SQL Cockpit version 1.0!

## What's New

This major release includes:

- **Enhanced Dashboard**: A completely redesigned user interface with improved navigation
- **Better Performance**: Optimized sync operations with configurable batch sizes
- **New Integrations**: Support for MCP Server and REST API enhancements
- **Improved Documentation**: Comprehensive guides for users and developers

## Getting Started

To get started with SQL Cockpit v1.0:

1. Download the latest release from our [GitHub repository](https://github.com/your-org/sql-cockpit/releases)
2. Follow the [installation guide](../../getting-started.md)
3. Check out the [user guide](../../user/index.md) for detailed instructions

## Upgrade Notes

If you're upgrading from a previous version, please review the migration guide for important changes.
